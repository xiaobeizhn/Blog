欢迎来到我的博客！这是第一篇文章。

这个博客使用 [Astro](https://astro.build/) 构建，部署在 Netlify 上，源码托管在 GitHub。文章用 Markdown 写成，可以在网站后台用实时编辑器创建和编辑。

## 特性

- 极简清爽的阅读体验，支持深色模式
- 用 Markdown 写作，所见即所得
- 后台实时编辑与预览，保存即发布
- 标签、RSS、SEO 全部就绪

## 代码示例

```javascript
function greet(name) {
  return `你好，${name}！`;
}

console.log(greet('世界'));
```
