博客后台使用 Cherry Markdown 编辑器，支持分屏实时预览。这篇文章介绍基本用法。

## 编辑器布局

编辑器分为左右两栏：

- **左侧**：Markdown 源码编辑区
- **右侧**：实时渲染预览区

输入内容时，右侧预览会即时更新。

## 常用语法

### 标题

用 `#` 号表示标题层级：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **加粗**：`**文字**`
- *斜体*：`*文字*`
- ~~删除线~~：`~~文字~~`

### 代码

行内代码用反引号：`console.log('hello')`

代码块用三个反引号：

```python
def hello():
    print("你好，世界！")
```

### 图片

可以直接**粘贴**或**拖拽**图片到编辑器中，图片会自动上传到仓库并插入链接。

也可以手动输入：

```markdown
![图片描述](/images/example.png)
```

### 表格

| 功能 | 快捷键 |
|------|--------|
| 加粗 | Ctrl+B |
| 斜体 | Ctrl+I |
| 链接 | Ctrl+K |

## 发布流程

1. 在后台点击「新建文章」
2. 编辑内容（frontmatter 会自动生成）
3. 点击「保存并发布」
4. 等待 1-3 分钟，Netlify 自动构建部署

文章保存后，会通过 GitHub API 直接写入仓库，触发自动重建。
